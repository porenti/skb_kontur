В папке solution имеются два каталога: 
	research
		task.md - задание, 
		train.tsv - начальные данные для обучения, 
		test.tsv - данные для прогноза,
		research.ipynb - файл с исследованием

	reproduction
		train.tsv - начальные данные для обучения, 
		predictions.tsv - данные для прогноза,
		main.ipynb - файл с моделью для воспроизведения предсказания
